建立git仓库

1. 把已有的项目代码纳入Git管理
$ cd 项目代码纳入Git管理
$ git init

2. 新建的项目直接用Git管理
$ cd 某个文件夹
$ git init your_project #会在当前路径下创建和项目名称同名的文件夹
$ cd your_project

git add -u # 把已经被git跟踪的文件的变更添加至暂存区

git log --all --graph -n2 --oneline

git help --web log

git cat-file -p xxxxxx

git add xx
git rm xx
git mv xx xxx

git reset --hard
